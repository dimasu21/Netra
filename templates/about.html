Tentang Kami

Apa itu NETRA?

NETRA adalah aplikasi web yang membantu Anda dan peneliti dalam proses screening literatur. Dengan menggabungkan teknologi OCR (Optical Character Recognition) dan algoritma Rabin-Karp, NETRA dapat mengekstrak teks dari dokumen fisik dan mencari kata kunci secara efisien.

Latar Belakang

Proses tinjauan pustaka seringkali memakan waktu yang lama, terutama ketika harus memeriksa relevansi buku atau jurnal fisik secara manual. NETRA hadir untuk mengatasi masalah ini dengan otomatisasi proses screening menggunakan algoritma pencarian string yang efisien.

Proyek ini dikembangkan untuk mendemonstrasikan implementasi praktis dari algoritma Rabin-Karp dengan pendekatan Rolling Hash.

Teknologi yang Digunakan

Flask (Python)

Backend framework untuk API dan routing

Tesseract OCR

Optical Character Recognition untuk ekstraksi teks dari gambar

Rabin-Karp Algorithm

Algoritma pencarian string dengan Rolling Hash

PyPDF2 & python-docx

Library untuk parse dokumen PDF dan DOCX

NETRA

Platform analisis dokumen cerdas berbasis OCR dan algoritma Rabin-Karp untuk membantu Anda dalam proses tinjauan pustaka.
