FAQ

Pertanyaan yang Sering Diajukan

NETRA membantu mempercepat proses screening literatur dengan menganalisis relevansi dokumen terhadap kata kunci riset Anda menggunakan algoritma Rabin-Karp. Anda dapat memproses puluhan dokumen sekaligus dan mendapatkan hasil dalam hitungan detik.

NETRA mendukung berbagai format file termasuk gambar (JPG, PNG) melalui fitur OCR, dokumen PDF, dan file Word (DOCX). Anda dapat mengupload dan menganalisis semua format tersebut secara langsung.

Algoritma Rabin-Karp menggunakan teknik rolling hash untuk mencari pola teks secara efisien. NETRA menggunakan algoritma ini untuk mencocokkan kata kunci riset dengan isi dokumen, menghasilkan persentase relevansi yang akurat.

Ya, keamanan data adalah prioritas kami. Dokumen yang Anda upload hanya diproses sementara untuk analisis dan tidak disimpan di server kami. Setelah analisis selesai, file akan otomatis dihapus.

Dengan fitur Batch Analysis, Anda dapat mengupload dan memproses hingga 20 dokumen sekaligus. Sistem akan menganalisis setiap dokumen dan menampilkan hasil relevansi dalam bentuk tabel yang mudah dibaca.

NETRA menyediakan paket gratis dengan batasan tertentu. Untuk penggunaan lebih intensif, tersedia paket berbayar dengan fitur tambahan seperti analisis batch unlimited dan prioritas processing.
